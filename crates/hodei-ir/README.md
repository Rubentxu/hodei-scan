# hodei-ir

Intermediate Representation (IR) core types for hodei-scan.

## Overview

This crate provides the core data structures and types for representing code analysis facts in a type-safe, language-agnostic format.

## Key Components

- **Fact Types**: Type-safe representations of code analysis findings
- **Core Types**: Atomic facts, IDs, locations, and provenance
- **Schema**: Versioned IR schema for serialization

## Usage

```rust
use hodei_ir::Fact;
```
